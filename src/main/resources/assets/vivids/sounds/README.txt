FALTAN LOS ARCHIVOS DE AUDIO REALES.

No puedo generar sonidos (.ogg) - eso queda fuera de mis capacidades.
Este proyecto SI tiene los eventos de sonido registrados en Java
(ModSounds.java) y referenciados aqui en sounds.json.

Para completarlo, agrega archivos .ogg con estos nombres EXACTOS en
esta misma carpeta:

  vivid_idle.ogg
  vivid_hurt.ogg
  vivid_death.ogg
  vivid_greet.ogg
  ray_shoot.ogg
  ray_overheat.ogg

Puedes grabarlos tu mismo, usar freesound.org (licencia libre) o
generarlos con alguna herramienta de audio. El juego no crasheara si
faltan, pero no se escuchara nada hasta que los agregues.
