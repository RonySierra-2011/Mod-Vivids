Aqui van los archivos .nbt de estructuras (Campamentos Vivid, Templos
del Bano, Ruinas, Pozos, Cuevas especiales). Se generan facilmente
in-game usando un Structure Block en modo SAVE:
  1) Construye la estructura en el mundo.
  2) Colocala dentro del area de un structure block.
  3) Guardala con el nombre vivids:nombre_estructura.
  4) Copia el .nbt resultante (carpeta generated/.../structures) aqui.
Esta carpeta esta vacia intencionalmente: no puedo generar geometria
3D de estructuras Minecraft desde texto.
